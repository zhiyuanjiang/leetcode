* lc_2866

单调栈，计算前缀和（写出来了hhh）。
这题最简单就是直接枚举峰值，可惜时间复杂度是$n^{2}$。假设$S_{i}$以$i$为山峰的最大值，$L_{i}$为左侧山峰的最大值，$R_{i}$为右侧山峰的最大值。那么最终的答案为:
$$ans = max{L_{i} + R_{i} - V_{i}}$$

先看${L_{i-1}}$和$L_{i}$之间是否存在递推关系。
如果$V_{i} > V_{i-1}$，则有:
$$L_{i} = L_{i-1} + V_{i}$$

如果$V_{i} < V_{i-1}$，则有:
$$L_{i} = L_{k} + (i-k)*V_{i}$$
这里的$k$表示从$i$到$k$，第一个小于$V_{i}$的下标，这里就能用单调栈来进行处理。
