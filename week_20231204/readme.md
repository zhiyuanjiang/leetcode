该题可以使用枚举的思维，有个隐藏的枚举点—字母数最大为26，如果考虑一个子串是否符合要求，那么子串的长度一定为
1*k, 2*k, ..., 26*k的情况，那么我们可以对这个长度进行枚举。所以我们需要做的就是进行分组枚举（分组的原因是第一个限制条件，相邻字符大小不能超过2）。时间复杂度为26n。
